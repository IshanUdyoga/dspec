# Images Folder

Place your image files here with the following filenames:

| Filename        | Description                                       |
|-----------------|---------------------------------------------------|
| `logo.png`      | Dspec Garage logo (blue D + SPEC GARAGE)          |
| `engine.jpg`    | Car engine with wrenches (used as hero background)|
| `workshop.jpg`  | Workshop interior with tools on pegboard          |

These names are referenced directly in `index.html` and `css/style.css`.
Rename your files to match exactly or update the references in the HTML/CSS.
